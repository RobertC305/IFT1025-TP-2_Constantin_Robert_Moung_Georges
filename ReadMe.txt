Ce projet comprend 3 programmes exécutables:
- Server.jar
- IFT1025-TP2-client-main.jar
- IFT1025-TP2-clientFX.jar

Le serveur permet au clients de lire les cours diponibles dans le fichier cours.txt et de s'inscrire dans le document inscriptions.txt.
Le client a une version qui fonctionne dans le Command Prompt et une version avec une interface graphique.

IMPORTANT: Nous avons supposé que les fichiers cours.txt et inscriptions.txt seront dans le même dossier que les .jar, il faut donc s'assurer que s'est bien le cas sinon le serveur ne les trouvera pas. 

P-S: Nous avons préparé un dossier test qui contient les fichier .jar et les fichiers cours.txt et inscriptions.txt afin de les tester.
